git是常用版本管理工具。

### 初始化

进入目录
```shell
git init
```
或克隆网络目录

```shell
git clone https://gitee.com/jdzcn/html
```

### 提交暂存区

```shell
git add .
git add *
```

### 提交版本

```shell
git commit -m 'description'
gut commit -a
```

### 撤销修改

未暂存撤销

```shell
git checkout -- filename
```
已暂存撤销

```shell
git reset HEAD readme.txt
```

已提交撤销
```shell
git reset --hard HEAD^
```
### 远程管理

显示远程库连接
```shell
git remote -v
```

新建远程库连接
```shell
git remote add origin git@github.com:michaelliao/learngit.git
```
删除远程库连接
```shell
git remote rm origin
```

远程提交
```shell
git push
git push -all

```
