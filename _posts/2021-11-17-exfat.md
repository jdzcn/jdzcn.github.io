### Install
```
​​$ sudo apt-get install exfat-utils exfat-fuse
```
### Format
```shell
$ sudo mkfs.exfat /dev/sdX1
```
### Mount
```shell
sudo mount -t exfat /dev/sdc1 /media/exfat
```
### Fstab
```shell
UUID=***** /home/sb/movie exfat defaults 0 0
```
